Mundial 2026 - Fixture interactivo neutro

Archivos:
- index.html
- styles.css
- app.js

Cómo usar:
1) Abrí index.html en el navegador para probar la web.
2) Para publicarla, subí estos 3 archivos a un repositorio de GitHub.
3) Activá GitHub Pages desde Settings > Pages > Deploy from branch > main > /root.

Funciones:
- Grupos A-L.
- Partidos de fase de grupos.
- Resultados editables.
- Tablas automáticas.
- Clasificados y mejores terceros.
- Llave hasta la final.
- Botón para activar partidos.
- Descarga de recordatorios .ics para calendario.

Fuente del fixture:
- FIFA y ESPN, consultado el 10 de junio de 2026.
- Horarios convertidos a Uruguay (UTC-3) tomando como base los horarios ET publicados.

Nota:
Antes de lanzar una campaña pública, conviene revisar posibles cambios de horario o sede en la fuente oficial.


Corrección de datos generales: Mundial 2026 = 48 selecciones, 12 grupos y 104 partidos totales. La fase de grupos conserva 72 partidos.
